第一章 教育基础知识

第一节 教育的产生和发展

一、教育的含义（考频很低）

（一）“教育”一词的由来【单选】

只需要记住：

  • 一人：孟子
  • 一书：《孟子 尽心上》

（二）教育的概念【单选、辨析】

广义：包括社会、学校以及家庭教育（广泛）
狭义：学校教育（有目的、有计划、有组织地进行）

二、教育的基本要素（单选）

教育者（主导）、受教育者（学习的主体）、教育影响（纽带、中介）

三、教育的属性【重点、单选、辨析】

  • 本质属性（育人性）【区别于其他社会活动的根本特征，质的规定性】：教育是有目的培养人的社会活动（偶然发生、动物、本能都不属于教育）
  • 社会属性🔆🔆🔆：
    • 永恒性：人在教育在
    • 历史性：古今不同（内容、方法有差别）
    • 继承性：古今相同
    • 长期性：培养周期长
    • 相对独立性🔆🔆🔆：自身规律、不平衡

      独立性表现为有其自身规律；
      不平衡表现为其可以滞后于或者超前于政治经济的发展，例如我国的教育就明显滞后于政治经济的发展。
      教育必然会受到政治经济的影响

    • 生产性：潜在的投资（比如买课提升自己）

四、教育的功能【单选、辨析】

  • 作用对象：
    • 个体功能（本体功能）：个体社会化（自然人变为社会人）、个体个性化（独特、与众不同的人）
    • 社会功能（派生功能）：
      • 变迁（大，广泛）：例如政治、经济、文化等
      • 流动（社会成员）：区域、职业
  • 性质/作用方向：正向、负向功能
  • 呈现方式：显性（与计划一致）、隐形（计划之外的、非预期的）功能

五、教育的起源【单选】

  • 生物起源学说（利托尔诺沛，西 能）：教育源于动物界的生存本能➡️本能生利息

    第一个正式提出的
    不是人独有的，否定了社会性和目的性

  • 心理起源学说（孟禄）：源于儿童对成人的“无意识的模仿”➡️心理仿梦露

    是人独有的，但是也否定了社会性和目的性

  • 劳动起源说（米丁斯基，凯洛夫）：源于劳动过程中的社会生产需要和人的发展需要的辩证统一➡️米凯爱劳动

    目前认为最科学的
    是人独有的，同时肯定了社会性和目的性

六、教育的发展

（一）原始社会的教育【单选】

简单、劳动、无阶级

（二）古代社会的教育【单选】

  • 奴隶社会：
    • 庠：中国最早的学校
    • 六艺：礼乐射御书数（礼乐是核心）
    • 西周：学在官府、政教合一（阶级性）
    • 春秋：私学兴起
  • 封建社会：
    • 隋唐：科举制兴起
    • 宋朝：四书（孟子，中庸，大学，论语）五经（诗，书，礼，义，春秋）
    • 清末：废除科举制（1905年）
    • 七艺
      • 三科：文法，修辞，辩证法
      • 四学：算数，集合，天文，音乐
  • 考点归纳：
  • 古代教育的总体特征：
    • 古代产生了学校
    • 教育与生产劳动相脱离、非生产性
    • 阶级性（你有我无）、等级性（你有我优）
    • 道统性、专制性、刻板性、象征性
    • 学校教育初步发展，尚未形成复杂的结构体系

（三）近代社会的教育（口诀公义世法）【单选】

  1. 公立教育崛起
  2. 初等义务教育的普遍实施，德国
  3. 教育的世俗化
  4. 教育的法制化

（四）20世纪以后的教育（口诀：全民多现身）【单选、简答（二级）】

  • 终身化
  • 全民化
  • 民主化：平等、自由
  • 多元化：目标、形式、管理、内容、标准等等
  • 现代化：设备、手段

（二）创立阶段【单选】

  • 《大教学论》：教育学称为一门独立学科的标志，近代第一本教育学著作

3. 康德（口诀：康德首讲授）
4. 赫尔巴特（口诀：巴特去规范，1234）
  • 现代教育学之父，科学教育学之父，传统教育代表人
  • 《普通教育学》：标志着教育学成为独立、规范的学科正式诞生，现代第一本教育学著作
    （1）一原则：教育性教学原则
    （2）二基础：伦理学和心理学作为教育学的理论基础
    （3）三中心：教师中心、教材中心、课堂中心
    （4）四阶段：明了、联想、系统、方法
5. 杜威（口诀：杜威真现代，3中心4个即）
  • 进步教育代表人，现代教育代表人，实用主义哲学创始人
  • 《民族主义与教育》
    （1）新三中心论：儿童中心（学生中心）、经验中心、活动中心
    （2）教育的本质：教育即生活，教育即生长，教育即经验的改组
    （3）学校即社会
    （4）从做中学
6. 归纳
7. 卢梭（口诀：卢的爱儿很自然）
  • 《爱弥儿》：提倡自然主义教育
8. 洛克（口诀：绅士洛克在白板上画漫画）
  • 《教育漫话》：白板说、绅士教育
9. 斐斯泰洛奇（口诀：斐爱心理重劳动）（考的很少）
  • 慈爱的儿童之父
  • 西方教育史上第一个把教育与生产劳动相结合付诸实践的教育家

一、生产力与教育的关系【❣️❣️❤️❤️】【单选、简答】【一级简答题，只需写下面的内容即可】

生产力对教育的制约——决定教育发展水平【口诀：龟速构防守】

  1. 决定教育发展的速度和规模
  2. 制约着人才培养的规格（培养什么样的人）和教育结构（学校专业比例）
  3. 制约着教学的内容、方法、组织形式与手段
    教育对生产力的反作用——经济功能【口诀：两再一新促生产】
  4. 劳动力再生产的基本途径
  5. 科学知识再生产的最有效形式
  6. **进行技术创新**的一个重要手段

二、社会政治经济制度与教育的关系【❣️❣️】【单选、简答】【二级简答题，只需写下面的内容即可】

社会政治经济制度对教育的制约【口诀：体内性质两权一亩地】

  1. 社会政治经济制度的性质决定着教育的性质
  2. 社会政治经济制度决定教育的宗旨与目的（培养社会主义人才）
  3. 社会政治经济制度决定教育的领导权
  4. 社会政治经济制度决定受教育权
  5. 社会政治经济制度决定教育的内容（与政治有关的内容）和教育管理体制
    教育对政治经济制度的影响作用——政治功能【口诀：人鱼公主促政治】
  6. 教育培养出政治经济制度所需要的人才——主要途径
  7. 教育促进政治民主化
  8. 宣传统治阶级的思想意识，创造社会舆论
  9. 传播一定的社会政治意识型态，完成年轻一代的政治社会化

二、个体身心发展的一般规律【❣️❣️❤️❤️】【单选、简答】【非常爱考】【二级简答题，主要以单选题为主】【口诀：不断补顺差】

（一）顺序性【单选】

表现：由……到……，从……到……，先……后……
启示：循序渐进，拔苗助长

（二）阶段性【单选】

表现：不同阶段有不同特征，矛盾，任务（少年期，又称为“危险期”或“心理断乳期”）
启示：分阶段教学，有针对性

（三）不平衡性【单选】

表现：速度有快有慢（同一方面不同速，不同方面不同步）
启示：关键期（劳伦兹——印刻实验）

（四）互补性【单选】

表现：身身互补，身心互补
启示：扬长避短，涨善救失

（五）差异性【单选】

表现：群体差异，个体之间
启示：因材施教

###四、影响人身心发展的因素【❣️❣️❤️❤️】【单选、简答、辨析、材料分析】【非常爱考】

（一）遗传【单选】

（1）遗传素质是人的身心发展的生理前提，为人的身心发展提供了可能性
（2）遗传素质具有可塑性

（二）环境【单选】

（1）环境为个体发展提供了多种可能，包括机遇、条件和对象
（2）环境是人的身心发展的外部的客观条件，环境使遗传提供的发展可能性变成现实
（3）环境不决定人的发展。人在接受环境影响时不是消极的、被动的。人具有主观能动性。

（三）学校教育（主导）【单选，简答】【三级简答题】

  1. 原因【口诀：三有一训练，控制很全面】：
    （1） 学校教育是有目的、有计划、有组织地培养人的活动
    （2） 由专门训练的教师来进行，效果更好
    （3） 有效地控制、影响学生发展的各种因素
    （4） 给人的影响比较全面、系统和深刻
  2. 表现【口诀：严规范，塑个性】：
    （1）按照社会对个体的基本要求对个体发展方向做出社会性规范
    （2） 具有加速个体发展的特殊功能
    （3） 对个体发展的影响具有即时和延时的价值
    （4） 具有开发个体特殊才能和发展个性的功能

（四）个体的主观能动性【单选，辨析】

（1） 是推动人的身心发展的内在动力
（2）个体的实践活动是个体发展的决定性因素

第五节 教育制度【重点把握学校教育的类型和我国学制的沿革】

一、教育制度的概述【❣️】

（一）教育制度的概念【单选】【主要考察：1.地位；2.狭义概念】

广义：国民教育制度
狭义：学校教育制度（学制），各级各类学校的总体系
地位：核心和主体

（二）教育制度确立的依据【单选、简答】【二级简答题】

1、社会依据

（1）生产力发展水平和科学技术发展状况
（2）社会政治经济制度
（3）文化传统
（4）人口

2、人的依据（学生身心发展规律和年龄特征）
3、社学制本身的依据（本国学制的历史发展和外国学制的影响）

（三）教育制度在形式上的发展【单选】【考察频率较低】

1、前制度化教育
  1. 实体化教育（学校）
  2. 教育形式趋于稳定（学校教育形态）
2、制度化
  1. 正规教育
  2. 近代学校系统的出现，即学制
  3. 我国进入的标志：废科举、兴学堂（清末，1905年）
3、非制度化（理想装填，超越制度）
  1. 学习化社会
  2. 终身学习

二、学校教育制度的类型【❣️❣️❤️❤️】【单选】【国家+表现/评价】

（一）双轨制

  • 国家：欧洲（西欧）【英国】
  • 表现：学术轨（自上而下，资产阶级），职业轨道（自下而上，劳动人民）

（二）单轨制

  • 国家：美国

（三）分支制

  • 国家：前苏联、我国

三、现代教育制度的发展趋势【❣️】【单选】【只考红色四个方面】

（一）义务要延长

  • 起源：德国
  • 特点：强制性、免费性、普及性【口诀：强免普】

（二）普职要综合

（三）高等要大众

（四）终身要构建

  • 概念化/体系化：保罗朗格朗（终身教育之父）《论终身教育》
  • 最终形成：《教育-财富蕴藏其中》（四大支柱：学会认知，学会做事，学会共同生活，学会生存）【口诀：知事存活】

四、我国学制沿革【❣️❣️】【单选】【口诀：人（壬）颁布，鬼（癸）实施，丑男女有资本，长期嘘（戌）美国】

  1. 壬寅学制：第一个颁布的学制，没有实行（1902）
  2. 癸卯学制：第一个实施的现代学制，学习日本，“中学为体，西学为用”，“男女不同校”（1904）
  3. 壬子癸丑学制：第一个具有资本主义性质的学制，学习日本，男女同校，废除读经，学堂改学校（1912）
  4. 壬戌学制：实施时间最长的学制，学习美国，又叫六三三、新学制（1922）
