各类软件的远程访问

要把WSL绑定到宿主机上再进行远程访问。

netsh interface portproxy add v4tov4 listenport=[port] listenaddress=0.0.0.0 connecport=[port] connectaddress=[WSL IP]

搭建Gymnasium环境出现的问题

我是win11然后在anaconda里面搭了个虚拟环境，python版本是3.10.10，执行下面命令的时候会出现问题：

    pip install "gymnasium[all]"

安装到box2d-py的时候会报错说building wheel for box2d-py failed，搞了我好久，终于搞懂：

  1. 需要在本机上安装swig，这个的安装方法只需要去官网下载win的安装包，然后解压后系统环境的path里面配置解压后的swig.exe就行；
  2. 需要Microsoft Visual C++ 14.0 or greater，这个的话就去Microsoft官网下载MSVC就行，除了默认的，选择自己系统的SDK加上MSVC生成工具就行
